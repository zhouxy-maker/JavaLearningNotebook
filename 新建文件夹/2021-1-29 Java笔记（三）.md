# 方法的定义
- 概念： 实现特定功能的一段代码，可反复使用
- 定义语法：

```java
public static void 方法名称(){
	//方法主体
}
```
1. 方法定义在类内部，与mian方法并列
# 方法调用
方法名称();
调用方法时，优先方法内部代码，结束后，返回到方法调用处，继续向下执行。
# 方法参数
数据交互
调用方法时传入的数据
1. 形式参数
2. 实际参数

定义语法：

```java
public static void 方法名称(形式参数){
	//方法主体
}
```
调用语法

```java
方法名称(实际参数);
```
# 返回值

定义语法：

```java
public static 返回值类型 方法名称(形式参数列表){
	//方法主体
	return value；//返回值
}
```
